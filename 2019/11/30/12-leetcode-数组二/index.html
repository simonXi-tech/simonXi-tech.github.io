搜索旋转排序数组 II

  • 数组中存在重复元素，进行二分的时候，数组一定是一半有序的，一半无序。
    根据 有序部分的 start 和 end 判断目标值是否在有序部分，否则在无序部分。存在一种特殊情况
    如果 num[start] = nums[mid] = nums[end]，此时无法判断数组哪边是升序，哪边是降序。有解决这个问题，
    可以 破环这个状态，num[start] = nums[mid] 必定有二分的两部分数组中，有一部分值全部相等，
    而另一部分存在 目标值。
